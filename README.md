# 전자 요금 계산기

**주택용 전기요금 (저압) 계산법**
- 100kWh 이하 : kWh당 **60.7원**
- 100kWh 초과 : kWh당 **125.9원**
- 200kWh 초과 : kWh당 **187.9원**
- 300kWh 초과 : kWh당 **280.6원**
- 400kWh 초과 : kWh당 **417.7원**
- 500kWh 초과 : kWh당 **670.6원**

---

**요금 계산 예시**
- 150kWh 사용
  - 100kWh는 60.7원, 50kWh는 125.9원을 책정
  - 요금 : 186.6원
